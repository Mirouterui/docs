# 命令行参数

| 参数              | 默认值        | 解释                             |
| ----------------- | ------------- | -------------------------------- |
| --config          | ./config.json | 配置文件路径                     |
| --workdirectory   | ./            | 基础目录路径，在里面存放静态文件 |
| --databasepath    | ./database.db | 数据库路径                       |
| --autocheckupdate | true          | bool，是否检查更新               |

::: note

Docker版本默认禁用自动检查更新

 :::
